Preventable Deaths Tracker

A platform to explore concerns raised by coroners' to prevent future deaths

This website displays the data we 'scraped' from the Courts and Tribunals Judiciary Website to put Prevent Future Death reports, also known as PFDs, into a searchable and filterable table.

The data

The data in this table were obtained from the Courts and Tribunals Judiciary Website.

To capture the data, we run a web scrape which can be found in our Github repository.

We wrote about the web scrape in Nature, "How we learnt to stop worrying and love web scraping"

Research & dissemination

We are working on a number of research projects using these data, including an analysis of suicides and deaths from medicines and medical events.

We are writing monthly case reports in BMJ Evidence Based Medicine to highlight key lessons to prevent serious fatal and non-fatal harms.
